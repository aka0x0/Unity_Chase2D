[Unity 2D] An enemy chases the player: プレイヤーを追いかける敵
====

Overview

## Description

An enemy(red) chases the Player(white).
You can move the player with the cursor keys.
Enemy is instantinated from prefab.

敵（赤）が自機（白）を追いかけてきます。
自機はカーソルキーで操作できます。
敵はプレファブからインスタンス化して作成しています。
